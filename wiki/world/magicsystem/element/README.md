- 魔法元素是魔法师施放魔法的基础，魔法元素之于魔法师，就像箭矢之于弓箭，是必不可少的一部分。

## 认识魔法元素：什么是魔法元素？

- 魔法元素与自然元素有着千丝万缕的关系，却又完全不同。当处于自然元素浓郁的地方时，相应的魔法元素也会浓郁起来。并非是魔法元素从自然中而生，而是自然元素会吸引魔法元素的聚集。当然，自然元素富集程度并不是影响魔法元素含量的唯一因素。
- 当没有魔法师驱使的时候，魔法元素处于平和的状态，会缓慢流动，但除非浓度到达一定程度，仍然不可见。但当魔法师的精神与魔法元素所连接，开始驱使魔法元素之后，魔法元素就会暂时转化为拥有着魔法特性的实体元素，这被称为魔法元素的激发状态。
- 当魔法师的精神与魔法元素所链接，那么魔法元素可以暂时转化成真实存在的实体元素，这些转化而成的实体元素可以产生热量，可以推动物品，但与真实的自然元素仍有差别。这些实体元素只是一种假象，在失去魔法师精神连接后，失去了魔法师的精神力，就会缓慢变回相应的自由态的魔法元素。
- 魔法元素与自然元素最大的区别在于魔法元素拥有魔法特性，比如水元素就拥有清除，治愈，可塑性强等特性。
- 魔法元素还可以用于驱使真实存在的元素，比如你可以调动水元素吸附空气中的水聚集在一起，再驱散掉水元素，这样你就得到了一杯水。
- 在人们对魔法的研究尚未深入之时，人们发现魔法元素中可能还存在一种万能魔法元素，人们称之为“魔法精华”，也称“以太”。在魔法师驱使魔法时，这些以太可能加入其中，增强魔法的威力，当假象自然元素退还为魔法元素时，可能有一部分变为以太。

## 魔法元素的种类

- 在魔法元素中，被认为是最基础的元素是火、水、气、土，这也常常被认为是构成世界的四大元素。
- 在基础元素之外，也存在着其它元素，学者们将它们分为三类，基础元素组合而成的复合元素，与基础元素有一部分相似的衍生元素，完全独立于基础元素的独特元素。
- 复合元素是同时有多种元素组合而成的元素，他们在有着多个基础元素的元素特征的同时，也丢失了一部分元素特性。常见的复合元素有熔岩、沼泽、雾、沙。
- 衍生元素是由基础元素转化而来，有一些特性与基础元素重叠，也具有与基础元素完全不同性质的元素，常见的衍生元素有木、冰。金属、水晶、光、暗、电。
- 独特元素是性质完全独立于基础元素的元素，常见的独特元素有神圣、死灵、血肉、灵魂、空间、时间、油。
